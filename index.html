Atajos de teclado (Lista Desordenada)

  • Ctrl + s: guardar cambios en el archivo
  • Ctrl + c: copiar
  • Ctrl + v: pegar
  • Ctrl + x: cortar
  • Ctrl + z: deshacer
  • Ctrl + y: rehacer
