Sincronismo y asincronismo

Tengo una lista de tareas a completar y tienen un orden específico y cada tarea tiene un tiempo para su conclusión:

sincrónico

Una lista de tareas es sincrónica cuando realiza las tareas en orden esperando y espera que se concluya la anterior para realizar la siguiente.

asíncrono

Una lista de tareas as asíncrona cuando tiene realiza las tareas en un orden determinado, pero tiene permitido concluirlas luego de comenzar otras tareas, es decir que podra esperar la conclusion de una primer tarea que ha encargado y mientras tanto las tareas posteriores puede haber o no concluido.
