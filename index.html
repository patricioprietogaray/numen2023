Atajos de teclado (Lista Desordenada)

Las listas ordenadas permiten crear listas donde los items no están enumerados.

  • Ctrl + s: guardar cambios en el archivo
  • Ctrl + c: copiar
  • Ctrl + v: pegar
  • Ctrl + x: cortar
  • Ctrl + z: deshacer
  • Ctrl + y: rehacer

Atajos de Emmet (Lista ordenada)

Las listas ordenadas permiten crear listas donde los items están enumerados.

  1. Ctrl + k + u: comentar
  2. Ctrl + k + c: descomentar
  3. Ctrl + / (shift + 7): comentar y descomentar
  4. Alt + Shift + f: identar el código (no lo hace en manjaro linux)

Ejemplo de una diapo

En una etiqueta de párrafo escribiremos la gran mayoría del texto que llevará una página web. El texto que escribamos aquí podemos ponerlo en negrita, en cursiva o itálica, subrayado, resaltado, e incluso podemos imitar las letras pequeñas de un contrato.
Hasta podemos escribir fórmulas matemáticas como la de la relatividad de Einstein que es e=mc2.

Wiki

El término wikinota 1​ (palabra que proviene del hawaiano wiki, 'rápido')2​ alude al nombre que recibe una comunidad virtual, cuyas páginas son editadas directamente desde el navegador, donde los mismos usuarios crean, modifican, corrigen o eliminan contenidos que, habitualmente, comparten. No tiene por qué ser necesariamente un sitio en la web, puesto que hay wikis instalables para uso en el escritorio de un computador u ordenador personal o que pueden portarse en un llavero USB que lleven un entorno LAMP como, por ejemplo, XAMPP.[cita requerida]
